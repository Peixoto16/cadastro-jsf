Sobre - Cadastro de Pessoas

Sobre o Sistema

Este sistema foi desenvolvido como parte de um teste técnico para demonstrar conhecimentos em desenvolvimento Java EE com arquitetura em camadas.

Tecnologias Utilizadas

  • Backend: Java EE com EJB 3.2
  • Frontend: JSF 2.3 com PrimeFaces
  • Persistência: JPA 2.2 com Hibernate
  • Banco de Dados: PostgreSQL
  • Build: Maven

Arquitetura

O sistema foi desenvolvido seguindo uma arquitetura em camadas:

  • Domain: Entidades JPA, enums e objetos de valor
  • Application: Serviços EJB, DTOs e mappers
  • Infrastructure: Repositórios JPA
  • Presentation: Controllers JSF e páginas XHTML
  • Shared: Exceções customizadas e utilitários
  • Cadastro, edição, listagem e exclusão de pessoas
  • Cadastro, edição, listagem e exclusão de endereços
  • Validação de dados em múltiplas camadas
  • Interface responsiva com PrimeFaces
