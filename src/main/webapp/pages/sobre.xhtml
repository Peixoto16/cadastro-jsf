Sobre - Cadastro de Pessoas

Sobre o Sistema

Este projeto é uma aplicação web para cadastro e gerenciamento de pessoas, desenvolvida com a plataforma Jakarta. O sistema inclui funcionalidades de CRUD, controle de acesso baseado em papéis (`ADMIN` e `USER`) e exportação de dados, integração com api de cep para busca de endereço e testes unitários e de integração.

  • Desempenho: O sistema é rápido e preparado para lidar com muitos registros, usando consultas otimizadas e reaproveitamento de conexões BD.
  • Facilidade de manutenção: O código é bem organizado, separado em camadas, o que facilita futuras melhorias e correções.
  • Confiabilidade: O sistema faz validações em vários pontos e trata erros de forma centralizada, garantindo que o usuário sempre receba um retorno.
  • Segurança: Apenas usuários autorizados conseguem acessar as funções principais, e há proteção contra ações indevidas.
  • Testes: O projeto conta com diversos testes automatizados para regras de negócio e integração com o banco de dados.
  • Escalabilidade: O sistema pode rodar em mais de uma máquina ao mesmo tempo, se necessário, pois não depende de informações salvas em memória.
  • Documentação: README detalhado e comentários no código facilitam o entendimento por outros desenvolvedores.
